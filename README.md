# SpinningCube

[![Demo](https://i.imgur.com/lLLaj4s.png)](https://i.imgur.com/lLLaj4s.png)

## Installation

Clone the project

```bash
git clone https://github.com/ToxicSDev/SpinningCube.git
```

Go to the project directory

```bash
cd my-project
```

Compile the project

```bash
gcc cube.c -o cube -lm
```

Run the compiled program

```bash
./cube
```
